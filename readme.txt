A simple lightweight plugin to set a color to your mobile browsers adress bar.

== Description ==

This plugin allows you to set the Address Bar color for your website on Mobile Browsers. With the help of plugin you can match the color of the address bar with your brand color or the color of your websites color scheme. It adds the meta tags to the head of your website for Android, iOS and Windows Phones. You can set the color in the plugin settings.

### Supported Devices

* Android
* iOS
* Windows Phones

== Installation ==
1. Upload `address-bar-colorize.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` page in your `WordPress Dashboard`.

== Frequently Asked Questions ==
= Where can I change the color after I install the plugin? =
 
After installing the plugin you can edit the color at your `Dashboard > Settings > Address Bar Colorize`
